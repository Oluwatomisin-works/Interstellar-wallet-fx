# InterPockt

InterPockt is a lightweight multi-currency wallet system with deposits,
swaps, transfers, and transaction history.

## Features
- Create wallets
- Deposit funds
- Swap currencies (USDx, EURx, cNGN, cXAF)
- Transfer between wallets
- View balances & transactions
- Dark / Light / Blue themes
- Input validation (no alphabet amounts)
- No database required

## Tech Stack
- React (JavaScript)
- FastAPI backend
- In-memory storage

npm install
npm start
